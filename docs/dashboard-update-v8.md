## 2. 신설 '대안 추천' 탭 — 구조
- 탭 네비에 **'대안 추천'** 추가(준비/1~4일차 옆). 스크롤 연동 하이라이트 동일 적용
- 탭 내부는 3개 카드/아코디언 섹션(2-A / 2-B / 2-C)
- 데이터 소스: 신규 CSV 3종 + `index.html` FALLBACK 배열 (**내용 일치 검증**, 라인번호 아님)

| 섹션 | CSV | FALLBACK 배열명 |
|---|---|---|
| 2-A 일정 대안 | `data/alt_plan.csv` | `FALLBACK_ALT_PLAN` |
| 2-B 맛집 대안 | `data/alt_food.csv` | `FALLBACK_ALT_FOOD` |
| 2-C 종류 리스트 | `data/food_genres.csv` | `FALLBACK_GENRES` |

### 렌더 규칙
- **2-A/2-B**: 카드 리스트. `indoor=Y`(실내)면 🏠 뱃지, `N`이면 표시 없음. 축제/불꽃 관련 행은 축제색(`#c0432b`) 좌측 바
- **2-C**: 표. `like` 값으로 색 구분 — `O`=선호(녹 `#5c7d4f`), `△`=옵션(금 `#9a7b34`), `X`=제외(회색 `#b8b0a0`, 취소선)
